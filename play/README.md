# I5看视频,轻量级影视搜索    
![](./static/logo.png)


>轻量，是相对而言。相对于其他同类型的网站，I5做到了足够轻量。  
仅仅包含一个搜索功能，简单/直观。  
直接对接其他资源网，并过滤掉不健康/违规内容，化繁为简。
